## テストデータの作成

ELFパーサを作る上で必要になるのが、解析対象となるファイルの用意です。  
この本で実装するのは **64bit little-endian ELFパーサ** です。  
64bit ELFについては、ビルドした自分自身の実行ファイルを解析対象にできます。

```shell-session
$ cargo build
$ cargo run target/debug/elf-parser-in-rust-book
```

一方で、32bit ELFを用意しておくと、  
本書の後に32bit対応へ拡張するときの確認用データとして使えます。  
本編の実装では32bit ELFは解析対象外なので、ここで作る `main32` を入力にすると失敗します。

## 作り方

32bit ELFは、適当なCプログラムを用意してgccで生成します。

```c
#include <stdio.h>
#include <stdlib.h>

int main() {
    puts("Hello, world!");
    
    return EXIT_SUCCESS;
}
```

```shell-session
$ gcc -m32 main.c -o main32
```

もし `fatal error: bits/libc-header-start.h` のようなエラーメッセージが出た場合は、 `gcc-multilib` パッケージをインストールしてください。
